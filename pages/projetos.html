MEUS PROJETOS

AuxiliaMed

Otimização do acesso por meios digitais á informações médicas de modo eficaz, menos burocrático e de maneira confiável, pensando especialmente nos problemas que as pessoas da terceira idade enfrentam. Criado uma plataforma web e mobile.

Tecnologias usadas:

API REST e RPC, Banco de Dados Oracle, API do ChatGPT, integração de IA com Python
